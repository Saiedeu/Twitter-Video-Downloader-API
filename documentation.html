Authentication

API Key Required

All requests must include a valid API key for authentication. The API key can be provided in three ways:

1. URL Parameter

?api_key=YOUR_KEY

2. HTTP Header

X-API-Key: YOUR_KEY

3. POST Body

api_key=YOUR_KEY

Python 

import requests
import urllib.parse

API_KEY = 'XvApImAssUmAhmED1997rfgD'
BASE_URL = 'https://sidman-apis.unaux.com/xvid/v1'

def extract_video_url_param(twitter_url):
    """Extract video using URL parameter method"""
    params = {
        'url': twitter_url,
        'api_key': API_KEY
    }
    
    try:
        response = requests.get(BASE_URL, params=params, timeout=30)
        response.raise_for_status()
        
        data = response.json()
        
        if data['status'] == 'success':
            print(f"Found {data['total_variants']} video variants")
            print(f"Highest quality: {data['highest_quality']}")
            return data['video_urls']
        else:
            print(f"Error: {data['message']}")
            return None
            
    except requests.exceptions.RequestException as e:
        print(f"Request failed: {e}")
        return None

def extract_video_header(twitter_url):
    """Extract video using header authentication"""
    headers = {
        'X-API-Key': API_KEY
    }
    
    params = {'url': twitter_url}
    
    try:
        response = requests.get(BASE_URL, params=params, headers=headers, timeout=30)
        data = response.json()
        
        if data['status'] == 'success':
            return data['video_urls']
        else:
            print(f"Error: {data['message']}")
            return None
            
    except Exception as e:
        print(f"Request failed: {e}")
        return None

def extract_video_post(twitter_url):
    """Extract video using POST method"""
    data = {
        'url': twitter_url,
        'api_key': API_KEY
    }
    
    try:
        response = requests.post(BASE_URL, data=data, timeout=30)
        result = response.json()
        
        if result['status'] == 'success':
            return result['video_urls']
        else:
            print(f"Error: {result['message']}")
            return None
            
    except Exception as e:
        print(f"Request failed: {e}")
        return None

Common Issues & Solutions

Tweet not found or protected

The tweet may be deleted, private, or from a protected account. Ensure the URL is correct and the tweet is publicly accessible.

No video content detected

The tweet may not contain video content. The API only extracts videos, not images or GIFs.

API key authentication failed

Double-check your API key and ensure it's properly included in the request. The key is case-sensitive.
